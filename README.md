## Project Overview

This project is an **Admin CMS** for managing blog posts and categories.  
Users can:
- Create, edit, delete posts  
- Create, edit, delete categories  
- Only authenticated users can access the dashboard  
- Each post belongs to one category  
- Posts have status: Draft / Published  



---

## Tech Stack

**Frontend**
- Next.js + TypeScript  
- Tailwind CSS  
- Axios for API calls  
- React Icons  

**Backend**
- .NET 8 Web API  
- Entity Framework Core  
- SQL 
- JWT Authentication
